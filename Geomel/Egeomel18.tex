Une droite $\mathcal D$ admet pour système d'équations :
\begin{displaymath}
 \left\lbrace 
\begin{aligned}
 -3x +z +2 &= 0 \\
2x-y -3 &= 0
\end{aligned}
\right. 
\end{displaymath}
Déterminer un point $A$ (dans le plan $x=0$) et un vecteur $\overrightarrow u$ tels que $\mathcal D = A+\Vect(\overrightarrow u)$.
