Une droite $\mathcal D$ admet pour système d'équations :
\begin{displaymath}
 \left\lbrace 
\begin{aligned}
 4x -3z -1 &= 0 \\
y -1 &= 0
\end{aligned}
\right. 
\end{displaymath}
Déterminer un point $A$ (dans le plan $z=0$) et un vecteur $\overrightarrow u$ tels que $\mathcal D = A+\Vect(\overrightarrow u)$.
